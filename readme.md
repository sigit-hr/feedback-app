# Feedback Application

This project provides a simple full-stack setup (Backend + Frontend + PostgreSQL) that can be fully started using **Docker Compose**.

---

## How to Run

Navigate to the **docker** directory inside the project root and run:

```bash
docker-compose up -d
```

Frontend app will be available shortly at the following url:

http://localhost:4200

Now you can navigate to:

- Feedback form
- Feedback administration page

## Troubleshoot

Note: Before running make sure you don't have open ports on: 4200, 8080, 5432.
